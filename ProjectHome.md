# twaindotnet #

Twain Dot Net is a TWAIN library for the .net platform. It is based off the code from Thomas Scheidegger's excellent article on The Code Project: [.NET TWAIN image scanner](http://www.codeproject.com/KB/dotnet/twaindotnet.aspx)
